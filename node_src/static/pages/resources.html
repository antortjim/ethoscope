Created on CURA for printing on UltiMaker 2+ using PLA as filament

All the files above as one zip folder - includes GCODES and STLs from Onshape

The easiest way to print is to download the gcode zip file and uncompressit on a SD card. Use the SD card directly on the printer. One can print 2 whole ethoscopes selecting files numbered 1,2,3 (excluding walls). Versions greater than 1.6 rely exclusively on printed parts (no screws and nuts needed) and print two small extra components compared to the published version: a camera holder that holds the camera in place and a light strip holder that holds the IR light. We also recommend using A4 mirror paper like this one to cover the lightbox. It will create more homogenous and stronger IR illumination. A PDF template to print on the BACK of those can be downloaded here.
